# 5. Database

The library provides utilities to enhance database interactions.

## `Seeder`

An abstract base class for database seeders that provides advanced seeding capabilities.

## `Migration`

An abstract class for database migrations that includes helper methods for schema manipulation.

## `PrefixedModel`

A base model class that allows you to easily add a prefix to your table names, which is particularly useful for package-specific tables.

### Example Usage

```php
use Squipix\Support\Database\PrefixedModel;

class User extends PrefixedModel
{
    protected $prefix = 'auth_';
    // Table name will be 'auth_users'
}
```
