# 6. Routing

The library provides a fluent way to register routes through the `RouteRegistrar`.

## `RouteRegistrar`

The `RouteRegistrar` class allows you to define route groups and register routes with a clean, object-oriented syntax.

### Example Usage

```php
use Squipix\Support\Routing\RouteRegistrar;

class MyRouteRegistrar extends RouteRegistrar
{
    public function map()
    {
        $this->group(['prefix' => 'api'], function() {
            $this->get('users', 'UsersController@index')->name('users.index');
        });
    }
}
```
