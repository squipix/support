# 1. Installation

You can install the package via composer:

```bash
composer require arcanedev/support
```

## Setup

The package will automatically register itself using Laravel's package discovery.

If you want to manually register the service provider, add the following line to your `config/app.php` (for Laravel 10 and below) or `bootstrap/providers.php` (for Laravel 11+):

```php
Squipix\Support\Providers\ServiceProvider::class,
```
