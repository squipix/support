# 7. Stubs

The `Stub` class is a powerful utility for generating files from template "stub" files. It supports dynamic placeholder replacements.

## Basic Usage

Placeholders in your stub files should be formatted as `$PLACEHOLDER$`.

### Example Stub File (`class.stub`)

```php
namespace $NAMESPACE$;

class $CLASS$ {}
```

### Rendering a Stub

```php
use Squipix\Support\Stub;

$stub = Stub::create('path/to/class.stub', [
    'NAMESPACE' => 'App\\Models',
    'CLASS'     => 'User',
]);

echo $stub->render();
```

## Saving a Stub

You can also save the rendered content directly to a file:

```php
$stub->saveTo('app/Models', 'User.php');
```
