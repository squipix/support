# 2. Usage

The `arcanedev/support` library provides a set of tools and helpers to streamline Laravel package and application development.

## Core Features

- **Helpers**: Useful global functions for Laravel version checking and routing.
- **Service Providers**: Enhanced base providers for easier package registration (assets, config, migrations, etc.).
- **Database Utilities**: Base classes for Seeders, Models with table prefixes, and enhanced Migrations.
- **Routing**: Fluent route registration helpers.
- **Stubs**: A flexible Stub class for dynamic file generation.

Explore the specific documentation sections for detailed information on each component.
