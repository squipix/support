# 3. Helpers

The library provides a few global helper functions.

## `laravel_version`

Get the current Laravel version or check if it matches a specific version.

```php
// Get version
$version = laravel_version(); // e.g., "12.0.0"

// Check version
if (laravel_version('12.')) {
    // Logic for Laravel 12
}
```

## `route_is`

Check if the given route(s) match the current route. This is a wrapper around `Router::is()`.

```php
if (route_is('home')) {
    // Current route is 'home'
}

if (route_is(['home', 'profile.*'])) {
    // Current route matches home or any profile route
}
```
